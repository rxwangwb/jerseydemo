# 用Jersey构建RESTful服务

## 环境说明

* jdk版本：jdk 1.8
* tomcat版本：7.0
* jersey版本：2.26
* javax版本：2.1
* IDE:IntelliJ IDEA 2017.2.4

## 框架说明：

* 基于maven搭建的ssm框架(spring+struts+mybatis)；
* 日志整合为Log4j2；
